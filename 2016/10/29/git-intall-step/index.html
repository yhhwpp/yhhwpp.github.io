第三步，是设置处理 line endings的方式。
在文本处理中，CR/LF 是不同操作系统上使用的换行符：

  • Dos 和windows 采用回车 + 换行 CR/LF 表示下一行
  • Unix / Linux 采用换行符LF表示下一行
  • OS 则采用CR表示下一行

CR 与 LF 区别：

  • CR 用符号 \r 表示，十进制 ASCII 代码是13，十六进制代码为 0x0D
  • LF 使用 \n 符号表示，ASCII代码是10，十六制为0x0A

这里我选择第一项，将提交时候 windows 的 CRLF 会自动转化转化为Unix的 LF，
