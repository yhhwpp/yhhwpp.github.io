我的Atom插件集合（长期更新）

发表于 2017-01-18   |   分类于 前端   |   阅读次数

从Sublime Text转向Atom有三个月的时间，发现Atom的插件比Sublime Text要好使的多。并且有大名鼎鼎的Github开发并开源，果断拥抱之。

本人依着宁需勿滥的原则，列出我的常用插件列表，仅供参考。

hyperclick, js-hyperclick

这个两个插件配合使用，可以实现类似于IDE的Ctrl + click，跳转到变量函数声明或者定义的地方，墙裂推荐！

hyperclick链接

js-hyperclick链接

atom-react

可针对jsx文件自动缩进，代码提示，代码片段，格式化等快捷键，React开发者必备。链接

atom-beautify

代码美化插件，支持大多数编程语言，前端开发必备。链接

emmet

怎能少了大名鼎鼎的emment，快速开发html,css的前端必备。链接

highlight-selected

Atom的选中效果不是很明显，推荐用这个插件，选中时，相同的代码会自动高亮，灰常方便。链接

docblockr

快速书写注释的快捷插件。链接

autocomplete-paths

自动路径提示。写 import 模块的时候，灰常好使。链接
